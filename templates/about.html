Movie Review Classifier
  • Movie Review Classifier is based on NLP and Machine Learning in which you can enter your movie review and click on submit review.
    After that your review with the prediction will be shown to you. You will also get Two options here which are correct prediction and incorrect prediction. If you select one of these options your review will be stored in MySQL database and now model training will be done on this new Data. You will also get an option to submit another review.
  • Used NLP Preprocessing Techniques Like Stopwords and HashingVectorizer.
    Used Online Algorithms and out-of-core learning for bigger datasets for eg, SGD Classifier to train the model.
  • Build this web app using Flask, HTML, CSS, Bootstrap and used MySql as Database.
